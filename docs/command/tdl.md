## tdl

Telegram Downloader, but more than a downloader

### Examples

```
tdl -h
```

### Options

```
  -h, --help           help for tdl
  -n, --ns string      namespace for Telegram session
      --proxy string   proxy address, only socks5 is supported, format: protocol://username:password@host:port
```
